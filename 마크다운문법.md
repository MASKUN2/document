# 마크다운 문법 
C23113 U231113

* 띄어쓰기 : 여러번 `space`를 써도 1번으로 인식함


* 줄바꿈 : 마지막 문장에서 `space` 2개 + `enter`  


* 문단 : `enter` 2번


* 순서리스트 : `1. 2. 1) 2)`


* 무순서리스트 : `* - +`  수준은 `tap`

* 코드 표시 : ` `` ` 또는 블록은 ` ``` ` 두번으로 함

*  인용 : ` > , >>, >>>`

* 체크박스 : 
- [ ] 아메리카노
- [x] 카페라떼


* 이미지 삽입 : `![제목](URL)` 
* 링크 : `[제목](URL)`

* 표 :
